# Hardware Campaign Example

Design Prompt:
Build a single-page landing page for a premium hardware product using a campaign-grade, product-first composition rather than a generic SaaS layout.

Visual Thesis:
The page should feel like a cinematic object reveal translated into the web. The product is the protagonist. Copy, UI, and supporting sections exist to frame the object, not compete with it.

Composition Rules:
- Use asymmetry and negative-space dominance.
- Let the hero object carry roughly two-thirds of the hero's visual weight.
- Avoid trapping the product in a standard card or dashboard shell.
- Alternate between sparse reveal moments and compressed technical proof moments.

Material Hierarchy:
- Primary: matte black or industrial shell surfaces
- Secondary: optical reflections, smoked glass, restrained support panels
- Tertiary: one accent color used only for intelligence, active state, or optical energy

Section Rhythm:
1. Object Reveal
2. Precision Strip
3. Immersive Proof
4. Capability Modes
5. Materials / Engineering Detail
6. Reserve / CTA

Key Constraints:
- Do not default to a three-card feature row under the hero.
- Do not let every section use the same card treatment.
- Do not make the page feel like B2B SaaS with a hardware skin.
