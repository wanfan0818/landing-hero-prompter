# Hardware Campaign Rules

Use this file when the product is premium hardware or near-future consumer technology:

- AR / XR glasses
- wearables
- robotics
- automotive interfaces
- audio hardware
- premium devices with strong industrial design

This pattern is different from generic SaaS and different from soft editorial branding.
It should feel like a launch film translated into a landing page.

## Core Shift

Do not describe the page as a set of modules.
Describe it as a sequence of reveal moments built around a hero object.

## Hero Principles

For hardware-first pages:

1. The product is the protagonist.
2. The hero should not read as a standard left-text right-image split.
3. The product should feel oversized, cropped, suspended, or staged.
4. The product media should not be trapped in a normal card.

Useful language:

- `product-as-sculpture`
- `suspended in a dark optical chamber`
- `campaign-grade object reveal`
- `oversized product crop`
- `resting on a reflective pedestal`
- `lit with controlled edge highlights`

## Layout Principles

Prefer:

- asymmetry
- negative-space dominance
- one dominant visual force per section
- sparse luxury spacing
- compressed technical bands between larger visual moments

Avoid:

- equal two-column balance everywhere
- repeated card grids in every section
- symmetric modular SaaS rhythm
- dense dashboards unless the product is explicitly enterprise software

## Section Strategy

Hardware pages usually work best with this pacing:

1. Object Reveal
2. Precision / Spec Strip
3. Immersive Proof
4. Capability Modes
5. Materials / Engineering Detail
6. Reserve / Buy / CTA

Do not force a generic `features grid -> testimonials -> pricing -> footer` sequence unless the reference clearly requires it.

## Product Media Rules

The product image or video should often:

- bleed outside neat containers
- use controlled reflections
- sit against black or deeply restrained backgrounds
- carry more emotional weight than the copy

If AR or UI overlays are shown, they must feel:

- diegetic
- sparse
- precise
- integrated into the environment

Avoid floating app cards that make the page feel like mobile UI marketing.

## Typography Rules

For hardware campaigns, typography should be restrained.

- headline: large, controlled, high-confidence
- emotional line: optional serif italic or softer contrast phrase
- body: quiet and compact
- specs: small technical labels, often monospace

The copy should feel expensive, not verbose.

## Material Hierarchy

Good default stack:

- Primary: matte black, smoked metal, soft shadow, low-noise surfaces
- Secondary: lens reflection, optical glass, translucent support surfaces
- Tertiary: one accent color used only for intelligence, active state, or optical energy

If the accent is cyan, green, or red, keep it rare.
Accent overuse immediately cheapens the page.

## Section Rules

### Object Reveal

Make the product large, singular, and dominant.
Keep copy short.
Do not crowd the hero with too many badges, links, or stats.

### Precision Strip

Replace generic feature cards with a compressed technical band when possible.
Use large values, small labels, thin dividers, and a luxury instrumentation feel.

### Immersive Proof

Use full-bleed environmental media to show the product in context.
This section should prove the fantasy.

### Capability Modes

Use cinematic panels or alternating compositions instead of simple feature cards.
Each mode should feel like a use-case vignette.

### Materials / Engineering Detail

Use macro crops, close-up details, or exploded information sparingly.
This is the section that establishes authority.

### Reserve / CTA

End with ritual, not clutter.
The final CTA should feel calm, inevitable, and premium.

## Anti-Patterns

- putting the hero product inside a generic rounded rectangle
- using three feature cards immediately after the hero by default
- making every section look like a dark dashboard
- treating AR overlays like floating app screenshots
- repeating glass cards so often that the whole page becomes a component catalog
- using too much copy for a premium hardware brand
